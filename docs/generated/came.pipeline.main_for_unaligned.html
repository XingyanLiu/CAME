came.pipeline.main_for_unaligned

came.pipeline.main_for_unaligned(adatas, vars_use, vars_as_nodes, df_varmap, df_varmap_1v1=None, scnets=None, dataset_names=('reference', 'query'), key_class1='cell_ontology_class', key_class2=None, do_normalize=False, n_epochs=350, resdir=None, tag_data=None, params_model={}, params_lossfunc={}, n_pass=100, batch_size=None, plot_results=True, norm_target_sum=10000.0, save_hidden_list=True, save_dpair=True)

Run the main process of CAME (model training), for integrating 2 datasets of unaligned features. (e.g., cross-species integration)

Parameters

  • adatas (AnnData) – A pair of sc.AnnData objects, the reference and query raw data

  • vars_use (Sequence[Sequence]) – a list or tuple of 2 variable name-lists. for example, differential expressed genes, highly variable features.

  • vars_as_nodes (list or tuple of 2) – variables to be taken as the graph nodes

  • df_varmap (DataFrame) – pd.DataFrame with (at least) 2 columns. relationships between features in 2 datasets, for making the adjacent matrix (vv_adj) between variables from these 2 datasets.

  • df_varmap_1v1 (None, pd.DataFrame; optional.) – dataframe containing only 1-to-1 correspondence between features in 2 datasets, if not provided, it will be inferred from df_varmap

  • scnets (Optional[Sequence[spmatrix]]) – two single-cell-networks or a merged one

  • dataset_names (Sequence[str]) – a tuple of two names for reference and query, respectively

  • key_class1 (str) – the key to the type-labels for the reference data, should be a column name of adatas[0].obs.

  • key_class2 (Optional[str]) – the key to the type-labels for the query data. Optional, if provided, should be a column name of adatas[1].obs.

  • do_normalize (bool) – whether to normalize the input data (the they have already been normalized, set it False)

  • n_epochs (int) – number of training epochs. A recommended setting is 200-400 for whole-graph training, and 80-200 for sub-graph training.

  • resdir (Union[Path, str, None]) – directory for saving results output by CAME

  • tag_data (Optional[str]) – a tag for auto-creating result directory

  • params_model (dict) – the model parameters

  • params_lossfunc (dict) – parameters for loss function

  • n_pass (int) – number of epochs to skip; not backup model checkpoints until n_pass epochs.

  • batch_size (Optional[int]) – the number of observation nodes in each mini-batch, based on which the sub-graphs will be used for mini-batch training. if None, the model will be trained on the whole graph.

  • plot_results (bool) – whether to automatically plot the classification results

  • norm_target_sum (Optional[float]) – the scale factor for library-size normalization

  • save_hidden_list (bool) – whether to save the hidden states for all the layers

  • save_dpair (bool) – whether to save the elements of the DataPair object

Returns

outputs

Return type

dict
