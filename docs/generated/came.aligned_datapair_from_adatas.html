came.aligned_datapair_from_adatas¶

came.aligned_datapair_from_adatas(adatas, vars_feat, vars_as_nodes=None, oo_adjs=None, dataset_names=('reference', 'query'), **kwds)¶

Build AlignedDataPair object from a pair of adatas

Parameters

  • adatas (Sequence[AnnData]) – a list or tuple of 2 sc.AnnData object.

  • vars_feat (a sequence of variable names) – a name-list of variables that will be used as (cell) node features. for example, names of differentail expressed genes (DEGs), highly variable features.

  • vars_as_nodes (a sequence of variable names, optional.) – a name-list of variables that will be taken as nodes in the graph for model training. if None (not provided), it will be the same as vars_feat

  • oo_adjs (Optional[Sequence[spmatrix]]) – a sequence of (sparse) adjacent matrices of observations. for example, the single-cell network within each dataset.

  • dataset_names (Sequence[str]) – dataset names, for example, (‘ref’, ‘que’)

Returns

dpair

Return type

AlignedDataPair

Examples

>>> dpair = datapair_from_adatas(
...     [adata1, adata2],
...     vars_feat,
...     dataset_names = ['reference', 'query']
...     )
