came.pipeline.gather_came_results¶

came.pipeline.gather_came_results(dpair, trainer, classes, keys, keys_compare=None, resdir='.', checkpoint='best', batch_size=None)¶

Packed function for pipeline as follows: 1. load the ‘best’ model weights 2. get the predictions for cells, including probabilities (from logits) 3. get and the hidden states for both cells and genes 3. make a predictor
